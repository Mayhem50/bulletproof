02

How it works

Bulletproof teaches your AI agent to think like a senior backend engineer.

1

Teach your project

Run /teach once. It scans your codebase, asks about your stack and constraints, and saves context to .bulletproof.md.

2

Use skills as needed

Type a slash command like /fortify or /schema. The agent analyzes your code against real-world best practices and produces an actionable report.

3

Audit everything

Run /audit for a comprehensive health check across all 13 domains. Get a scored report with prioritized fixes — from P0 critical to P3 nice-to-have.

Backed by deep reference material

Every skill is supported by 7 reference documents — over 5,000 lines of dense, actionable backend engineering knowledge covering resilience patterns, data design, API conventions, observability, security, concurrency, and testing.
